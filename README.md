# 🧮 Multi-Calculadora com Streamlit

Uma aplicação web interativa que reúne diversas ferramentas de cálculo em um só lugar. Desenvolvida em Python com a biblioteca Streamlit.

## 🚀 Funcionalidades

O aplicativo conta com 8 modalidades de cálculo selecionáveis na barra lateral:

* **Operações Básicas:** Soma, Subtração e Multiplicação com suporte dinâmico para calcular de 2 até 100 números simultaneamente.
* **Divisão:** Divisão simples com tratamento de erro para evitar divisão por zero.
* **Cálculo de IMC:** Calcula o Índice de Massa Corporal e classifica o resultado (Abaixo do peso, Peso normal, Sobrepeso ou Obesidade).
* **Média Escolar:** Calcula a média de 4 notas e exibe o status do aluno (Aprovado, Recuperação ou Reprovado).
* **Desconto:** Descobre o preço final de um produto aplicando uma porcentagem de desconto.
* **Conversor de Moeda:** Converte valores de Dólar (USD) para Real (BRL) baseado em uma cotação fixa.

## 🛠️ Tecnologias Utilizadas

* **Python** (Linguagem base)
* **Streamlit** (Framework para a interface web)

## 📦 Como Executar o Projeto

### Pré-requisitos
Você precisa ter o **Python** instalado em sua máquina.

### Passo a Passo

1. **Clone o repositório:**
   ```bash
   git clone https://github.com
   cd seu-repositorio
   ```

2. **Instale o Streamlit:**
   ```bash
   pip install streamlit
   ```

3. **Execute a aplicação:**
   ```bash
   streamlit run seu_arquivo.py
   ```

4. **Acesse no navegador:**
   O Streamlit abrirá automaticamente uma aba no seu navegador padrão no endereço `http://localhost:8501`.
